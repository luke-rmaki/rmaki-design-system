## Variants

Variants are produced by adjusting the lightness value using calc.

Add as needed

### Function to calculate variants

```typescript
/**
 *
 * @param lightness number
 * @param variant string
 * @param step number
 * @returns number
 */
export function calculate_variant_lightness(
  lightness: number,
  variant: string,
  step: number
): number {
  const multiplier: number = step * 10;
  const percentage = (lightness / 100) * multiplier;

  if (variant === `dark`) {
    return lightness - percentage;
  }
  return lightness + percentage;
}
```

### Dark variants

#### Dark one

Subtract 10% from lightness value

```css
--color-dark-one: hsl(h, s, calc(l - 10%));
```

### Light variants

#### Light one

Add 10% to lightness value

```css
--color-dark-one: hsl(h, s, calc(l + 10%));
```
